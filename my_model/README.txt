PLACE YOUR TEACHABLE MACHINE MODEL FILES HERE:

1. model.json
2. metadata.json  
3. weights.bin (or similar weight files)

To get these files:
1. Go to https://teachablemachine.withgoogle.com/
2. Open your image classification project
3. Click "Export Model"
4. Choose "Tensorflow.js" format
5. Download and extract the files
6. Copy the files to this folder

Once you add these files, refresh the website and click "Start Camera"!
